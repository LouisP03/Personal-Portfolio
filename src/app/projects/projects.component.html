TeachCode
2023 - 2024

TeachCode is a platform developed entirely by me, as part of my Final Year Project module at University of Birmingham. It is a full-stack JHipster web application that allows users to register as a teacher or student, and host/participate in programming tasks. A teacher can create tasks with sample code/expected output/chosen programming language, start this task, and view submissions and analytics for their task. Students are able to join using a unique room code when the task is started, write code, run the code, and submit it.
